Installing
---

- Step 1 :
```
git clone https://github.com/maisheneka/deploy-system-use-iac.git
```
```
cd deploy-system-use-iac
```

- Step 2 :
```
cd path-folder
```

- Step 3 :
```
terraform init
```

- Step 4 :
```
terraform plan
```

- Step 5 :
```
terraform apply -auto-approve
```

- Step 6 :
```
terraform destroy -auto-approve
```
